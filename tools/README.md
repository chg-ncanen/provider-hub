# Tools

Reusable utilities and libraries for local use (not deployed).

## Structure

- `team/` — Shared tools available to all team members (organized by team area: provider, pde, web)
- `user/` — Individual user tools and utilities

## How to Add a Tool

1. Create a directory under `team/<team-area>/` or `user/<your-name>/`
2. Include source files, dependency specifications, and documentation
3. Submit a PR for review

## Available Tools

Check subdirectories for individual tool documentation.
