# AI Skills

Copilot and autonomous agent skills shared across the team.

## Structure

- `team/` — Shared skills available to all team members (organized by team area: provider, pde, web)
- `user/` — Individual user skills (can be opt-in to the team later)

## How to Add a Skill

1. Create a directory under `team/<team-area>/` or `user/<your-name>/`
2. Add a `SKILL.md` file with skill metadata (see [Skill Format](../../../CONTRIBUTING.md))
3. Include any supporting files (Python scripts, configs, etc.)
4. Submit a PR for review

## Available Skills

See `SKILL.md` files in subdirectories for individual skill documentation.
