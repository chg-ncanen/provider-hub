# APIs

OpenAPI specifications, SDKs, and integration examples.

## Structure

- `team/` — Shared APIs available to all team members (organized by team area: provider, pde, web)
- `user/` — Individual user APIs and integrations

## How to Add an API

1. Create a directory under `team/<team-area>/` or `user/<your-name>/`
2. Include OpenAPI spec (YAML/JSON), SDK examples, and documentation
3. Submit a PR for review

## Available APIs

Check subdirectories for individual API documentation.
