# MCP (Model Context Protocol) Servers

MCP server configurations and definitions.

## Structure

- `team/` — Shared MCP servers (organized by team area: provider, pde, web)
- `user/` — Individual user MCP servers

## How to Add an MCP Server

1. Create a directory under `team/<team-area>/` or `user/<your-name>/`
2. Include MCP configuration, server definition, and documentation
3. Submit a PR for review

## Available MCP Servers

Check subdirectories for individual MCP server documentation.
